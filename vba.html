Создание ежедневной отчётности

Автоматический сбор данных, формирование и рассылка отчётов через Outlook
01

Макрос-комбайн для ежедневной отчётности. На главном листе — инструкция, колонки для подсчёта, список адресов рассылки и три кнопки, каждая из которых запускает свой процесс. Вся работа с данными происходит автоматически: файлы берутся с сетевой шары, обрабатываются и сохраняются обратно.

⚙️ Три кнопки — три процесса:
1

Обновить СПК

Забирает с сетевой шары 3 исходных Excel-файла (два от лаборатории контроля, один с количеством сваренных стыков) и форму отчёта со вчерашними данными. Определяет нужную дату, находит данные за позавчерашний день во всех файлах, заполняет форму и сохраняет готовый отчёт обратно на шару.

2

Отправить СПК

Берёт готовый отчёт с шары и отправляет его через Outlook всем адресатам из списка на главном листе. Автоматически подставляет тему письма, текст и подпись — остаётся только нажать одну кнопку.

3

Обновить СПК с динамикой

Нажимается раз в неделю по четвергам. Перебирает все отчёты на шаре за последние 7 дней, собирает с них данные и записывает в сводный файл, в котором хранится статистика за все дни.

Проверка отчётов сварки

Проверка стыков по журналу, допускам и % контроля + сохранение с подписью
02

Макрос для полной автоматической проверки сварочных отчётов. На главном листе — инструкция и поле для выбора проверяющего (от этого зависит подпись). По нажатию кнопки «Проверить отчёты» открывается диалог выбора папки — дальше всё происходит автоматически.

📥 Этап 1 — Импорт данных:
1

Импорт отчётов

Из выбранной папки импортируются все данные по сварным соединениям — каждый стык со всеми параметрами.

2

Подгрузка базы данных и журнала

Подключаются база данных сварщиков (допуски каждого: диапазон диаметров, материалы, толщины) и сварочный журнал (сваренные и несваренные стыки). Импортируются только релевантные данные — то, что относится к проверяемым отчётам.

✅ Этап 2 — Проверка каждого стыка:
📐

Размеры и материал

По изометрии, типу и номеру стыка сверяются размеры и материал со сварочным журналом. Совпадения подсвечиваются зелёным, расхождения — красным. При расхождении макрос предлагает корректные значения из журнала.

🪪

Клеймо — проверка допуска

Клеймо сварщика проверяется по базе данных допусков — имеет ли этот сварщик право варить данный диаметр, материал и толщину. Результат выводится в отдельном столбце с цветовой подсветкой.

%

Клеймо — оптимизация по % контроля

Макрос анализирует процент контроля по линии — например, при норме 10% контролируется 1 стык из 10. Если у клейма BZA4 сварено 7 стыков и 1 проконтролен, а у B81H — 10 стыков и тоже 1 проконтролен, логичнее поставить BZA4, чтобы лишний раз не назначать контроль. Если в отчёте стоит менее оптимальное клеймо — макрос предлагает замену в отдельном столбце: клеймо, которое подходит и по допуску, и по проценту контроля.

💾 Этап 3 — Результат и сохранение:
📋

Сводный лист с подсветкой

Все данные собраны на одном листе с цветовой подсветкой: размеры и материал — зелёный/красный; клеймо дублируется в нескольких столбцах — один для проверки допуска, другой для допуска + % контроля, и отдельный столбец с предложением оптимального клейма.

💾

UserForm и сохранение

На сводном листе есть кнопка, открывающая UserForm. В ней чекбоксами отмечаются внесённые корректировки (клеймо, способ сварки, размеры, материал). По нажатию «Сохранить отчёт» макрос вставляет подпись проверяющего (считывается с шары по выбору на главном листе), сохраняет исправленный отчёт на рабочий стол в папку «Проверенные», а в имя файла записывает, какие именно изменения были внесены.

Подбор клейм для наряд-задания

Автоподбор сварщиков по допускам и оптимальному % контроля
03

Приходит наряд-задание — список стыков, которые будут вариться в ближайшее время. Задача — подобрать для каждого стыка сварщика, у которого есть допуск на данный диаметр, толщину и материал, и при этом выбрать того, кто оптимально вписывается по проценту контроля, чтобы не назначать лишнюю проверку на линии.

⚙️ Две кнопки — два этапа:
1

Импорт данных

По нажатию первой кнопки импортируются база данных сварщиков и сварочный журнал. Затем для каждого стыка из наряд-задания макрос находит в журнале и записывает диаметр, толщину, материал и процент контроля линии. Если какой-то стык не найден в журнале — он помечается красным цветом.

2

Подбор клейм

По нажатию второй кнопки макрос для каждого стыка перебирает всех сварщиков из базы, отсеивает тех, у кого нет допуска на данные параметры, а из оставшихся выбирает того, кто лучше всего подходит по проценту контроля — чтобы его стык не стал лишним контрольным и линия не была переконтролирована. Если у всех сварщиков на линии процент контроля уже закрыт, макрос подбирает нового сварщика, который подходит по допуску на максимальное количество стыков с этой линии. Результат — готовое наряд-задание с проставленными клеймами.

Проверка сварочного журнала

Проверка всех клейм журнала на допуск по диаметру, толщине, материалу и дате
04

Макрос для полной проверки сварочного журнала. По нажатию одной кнопки импортируется весь журнал и база данных сварщиков, после чего каждый стык проверяется — имеет ли указанное клеймо допуск на данный диаметр, толщину и материал, а также подходит ли сварщик по дате (действителен ли его допуск на момент сварки).

  • Импорт всего сварочного журнала и базы данных сварщиков
  • Проверка каждого стыка: диаметр, толщина, материал, дата допуска
  • Результат: допуск есть — зелёный, допуска нет — красный

Подбор техкарт

Автоподбор технологических карт по материалу, диаметру, толщине и способу сварки
05

Макрос работает прямо в открытом сварочном журнале. По нажатию кнопки импортируется перечень техкарт, затем макрос анализирует журнал — находит стыки, на которые нужно проставить техкарту, и подбирает подходящую карту из перечня в зависимости от материала, диаметра, толщины и способа сварки стыка.

  • Импорт перечня техкарт по нажатию одной кнопки
  • Анализ журнала: поиск стыков без техкарты
  • Подбор по материалу, диаметру, толщине и способу сварки
