Программа для отчётов прорабов

Создание отчётов, табелей, расстановок с базами данных и экспортом в Excel/1С
01

Десктопное приложение для создания ежедневных отчётов прорабов. Работает в WebView (Edge --app), состоит из двух частей: сборщик базы данных и основной интерфейс. Программа имеет функции автообновления и автоскачивания свежей базы данных при запуске.

🔧 Часть 1 — Сборщик базы данных:
📦

Сборка и публикация CSV-базы

Из сварочной книги, базы данных сотрудников и оспуленных изометрий собирается общая CSV-база (wb_main, opory, lr_base, welders) и загружается на сетевую шару. Сборка реализована на Python (без COM-зависимости от Excel).

💻 Часть 2 — Основной интерфейс:
🔍

Базы данных и поиск

Вкладки с базами: сварочный журнал, опоры, ЗРА, сварщики, база ЛР. По каждой базе работает поиск с мгновенной фильтрацией.

📋

Отчёт, расстановка, табель

Три рабочих вкладки: Отчёт — заполнение данных по сваренным стыкам, Расстановка — распределение персонала по объектам, Табель — учёт рабочего времени. При добавлении стыка, который уже сварен, выдаётся предупреждение.

👥

Редактор наборов и бригад

Создание и редактирование наборов прорабов-бригадиров и составов бригад. Наборы сохраняются локально и переиспользуются при заполнении отчётов.

📤

Экспорт и отправка

Экспорт в Excel с ключами каждого блока для последующего переноса данных из отчётов в 1С. Возможность отправить отчёт на сервер, прикрепив к нему изометрии.

⚡ Автоматизация:
🔄

Автообновление и автоскачивание

При запуске программа проверяет наличие новой версии на сетевой шаре и обновляется автоматически. Также автоматически скачивается свежая база данных. Если сеть недоступна — запускается с локальной базой.

Расселение сотрудников в общежития

Управление общагами, комнатами, жильцами и авторасселение с приоритетами
02

Программа для административно-хозяйственного отдела — полное управление расселением сотрудников по общежитиям. Четыре вкладки покрывают весь цикл: от создания общежитий до автоматического расселения прибывающих. Поддерживается импорт данных из Excel и 1С.

📑 Четыре вкладки:
🏠

Главная

Основная вкладка с общей сводкой по заселённости общежитий и быстрым доступом к ключевым функциям.

🏢

Общаги

Создание, удаление и просмотр общежитий. Для каждого — количество комнат, мест в комнатах и статус заселения: какие места свободны, какие заняты.

👤

Люди

Полный список жильцов — кто где живёт, в каком общежитии и какой комнате. Поиск и фильтрация по имени, общежитию, статусу.

📋

Расселение

Список сотрудников, прибывающих в ближайшее время. Кнопка авторасселения автоматически распределяет прибывающих по свободным местам во всех общежитиях, с приоритетом на семейные комнаты для женатых пар.

  • Импорт данных из Excel и 1С
  • Авторасселение с приоритетом семейных комнат для пар
  • Полная информация о заселённости: комнаты, места, свободные/занятые
